Keyboard shortcuts help you work faster. This reference covers all available shortcuts.

## General

| Shortcut | Action |
|----------|--------|
| <kbd>Ctrl</kbd> + <kbd>N</kbd> | New rack |
| <kbd>Ctrl</kbd> + <kbd>S</kbd> | Save/export |
| <kbd>Ctrl</kbd> + <kbd>O</kbd> | Open/import |
| <kbd>Ctrl</kbd> + <kbd>Z</kbd> | Undo |
| <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>Z</kbd> | Redo |
| <kbd>?</kbd> | Show keyboard shortcuts |

<Aside>
  On macOS, use <kbd>Cmd</kbd> instead of <kbd>Ctrl</kbd>.
</Aside>

## Selection

| Shortcut | Action |
|----------|--------|
| <kbd>Click</kbd> | Select device |
| <kbd>Shift</kbd> + <kbd>Click</kbd> | Add to selection |
| <kbd>Ctrl</kbd> + <kbd>A</kbd> | Select all devices |
| <kbd>Escape</kbd> | Clear selection |

## Device manipulation

| Shortcut | Action |
|----------|--------|
| <kbd>Delete</kbd> | Delete selected devices |
| <kbd>Backspace</kbd> | Delete selected devices |
| <kbd>Ctrl</kbd> + <kbd>C</kbd> | Copy selected devices |
| <kbd>Ctrl</kbd> + <kbd>V</kbd> | Paste devices |
| <kbd>Ctrl</kbd> + <kbd>D</kbd> | Duplicate selected devices |

## Movement

| Shortcut | Action |
|----------|--------|
| <kbd>↑</kbd> | Move device up 1U |
| <kbd>↓</kbd> | Move device down 1U |
| <kbd>Shift</kbd> + <kbd>↑</kbd> | Move device up 5U |
| <kbd>Shift</kbd> + <kbd>↓</kbd> | Move device down 5U |

## View

| Shortcut | Action |
|----------|--------|
| <kbd>Ctrl</kbd> + <kbd>+</kbd> | Zoom in |
| <kbd>Ctrl</kbd> + <kbd>-</kbd> | Zoom out |
| <kbd>Ctrl</kbd> + <kbd>0</kbd> | Reset zoom |
| <kbd>F</kbd> | Fit rack to view |

## Editing

| Shortcut | Action |
|----------|--------|
| <kbd>Enter</kbd> | Edit selected device |
| <kbd>Double-click</kbd> | Edit device |
| <kbd>Tab</kbd> | Select next device |
| <kbd>Shift</kbd> + <kbd>Tab</kbd> | Select previous device |

## Customising shortcuts

Keyboard shortcuts are not currently customisable. This feature is planned for a future release.

<Aside type="tip">
  Press <kbd>?</kbd> anywhere in the app to see this reference.
</Aside>
