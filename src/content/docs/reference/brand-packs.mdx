## Creating custom devices

Until brand packs are available, create devices manually:

1. Add a blank device to your rack
2. Set the correct U height
3. Name the device (e.g., "Dell R730")
4. Add notes with model details

<Aside type="tip">
  Custom devices can be saved as templates for reuse across multiple racks.
</Aside>
