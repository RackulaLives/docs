## Features

<CardGrid stagger>
  <Card title="Drag and Drop" icon="pencil">
    Build rack layouts visually. Drag devices, resize, and rearrange with ease.
  </Card>
  <Card title="Multiple Racks" icon="document">
    Plan entire data centre floors. Support for unlimited racks in a single project.
  </Card>
  <Card title="Export Anywhere" icon="add-document">
    Export as PNG, SVG, or PDF. Perfect for documentation and sharing.
  </Card>
  <Card title="Self-Host" icon="setting">
    Run on your own infrastructure. Docker image available, no external dependencies.
  </Card>
  <Card title="Privacy First" icon="approve-check">
    All data stays in your browser. No accounts, no tracking, no cloud storage.
  </Card>
  <Card title="Open Source" icon="github">
    MIT licensed. Contribute, fork, or audit the code on [GitHub](https://github.com/RackulaLives/Rackula).
  </Card>
